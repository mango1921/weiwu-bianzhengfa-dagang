作者简介

李达(1890—1966),字鹤鸣,湖南零陵人,中国马克思主义哲学家、教育家。1921 年中共一大代表,武汉大学校长。主要哲学著作有《社会学大纲》《唯物辩证法大纲》(主编)等。
